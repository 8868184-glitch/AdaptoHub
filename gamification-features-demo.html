â“ FAQ

Q: Do students need to opt-in to gamification?

A: No! Gamification is automatic. All students participate when they complete quizzes.

Q: Can I disable gamification?

A: Yes, by removing the gamification scripts from index.html if needed.

Q: How is progress saved?

A: All data is stored locally in each student's browser. No server required.

Q: Can I customize achievements?

A: Yes, modify thresholds in achievements.js and gamification.js.

Q: Is gamification optional for teachers?

A: Yes, it's fully optional. Teachers can ignore it if they prefer traditional grading.

Q: Does gamification affect grades?

A: No, gamification is purely motivational. Quiz scores remain the source of truth.
